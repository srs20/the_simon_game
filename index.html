Press A Key to Start

RULES:-

1. Simon will give the first signal. Repeat the signal by Pressing the same colour block.

2. Simon will add one more signal. Now you have to press the previous colour blocks with proper sequence and then the last signal given by Simon.

3. Continue playing as long as you can repeat these colour signals.

4. If you fail to repeat then Simon will give a RAZZ sound. You have to start over again
